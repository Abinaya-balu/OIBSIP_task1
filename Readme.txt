This is a simple web-based calculator that allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division.

Installation & Usage:

This calculator is a standalone HTML page and does not require additional setup.

    1.Download or clone this repository.
    2.Open the index.html file in a web browser.
    3.Start calculating!

How to Use:

1.Enter two numbers in the input fields.

2.Click on one of the four operation buttons:

    + for Addition

    - for Subtraction

    × for Multiplication

    / for Division

3.The result will be displayed immediately below the buttons.


Features:

Perform basic mathematical operations (+, -, ×, ÷)

User-friendly interface with easy input fields

Simple and clean design with responsive layout

Animated title effect for better UI experience

Technologies Used:

HTML: Structure of the web page

CSS: Styling for better UI/UX

JavaScript: Logic to perform calculations
